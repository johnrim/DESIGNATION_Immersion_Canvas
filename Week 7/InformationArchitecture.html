Brief

Information architecture is the art and science of organizing and labeling web sites, intranets, online communities and software to support usability and findability.

Card Sorting

Card sorting is a great, reliable, inexpensive method for finding patterns in how users would expect to find content or functionality.

Sorting Things Out: An Introduction to Card Sorting from Stephen Anderson

  • Card Sorting: A Definitive Guide

Different types
  1. Open: Participants create their own names for the categories.
  2. Closed: Participants are provided with a predetermined set of category names.
  3. Reversed: Users are given tasks and are asked to complete them navigating a collection of cards.

Tips
  1. Use 30-40 cards
  2. Select content that can be grouped
  3. Select terms and concepts that are meaningful to users
  4. Avoid generic terms like glossary that users user on every site

In-class Exercise

  1. Perform a content audit
  2. Create cards of the different pages, content, lessons
  3. Do card sorting using index cards or Post-it notes.
  4. From this card sorting, create a site map and navigation system.
