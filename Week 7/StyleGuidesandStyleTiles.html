What is a Style Guide?

A “Style Guide” is a term used to mean the set of visual elements that represent a brand. They include the name, logo and other identification devices such as color, type, format and graphic usage. When used consistently, these elements create a cohesive visual identity.

Style Guide Best Practices from Brad Frost

Key elements of every style guide:

  1. Logo: (logomark or logotype or both)
    • Usage
    • Do's/Don'ts
  2. Typography
    • Font family
    • Sizes
    • Do's/Don'ts
  3. Color: RGB/CMYK/Pantone values
  4. Brand attributes/Personality
  5. Image use
